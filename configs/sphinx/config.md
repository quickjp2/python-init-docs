# Sphinx configurations

## Common extensions

```python
extensions = [
    'sphinx.ext.autodoc',
    'sphinx.ext.githubpages',
    'sphinx.ext.todo',
    'sphinxcontrib.napoleon',
    'sphinx_autodoc_typehints',
]
```

## Setting the ReadTheDocs theme

```python
html_theme = 'sphinx_rtd_theme'
```
